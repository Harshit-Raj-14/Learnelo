```bash
git clone https://github.com/teamhanko/hanko-nextjs-starter.git
```
## Get the Hanko API URL

Get the API URL from [Hanko console](https://cloud.hanko.io/) and add it to `.env.local` file.

## Install dependencies

```bash
npm install
```

## Run the project

```bash
npm run dev
```
